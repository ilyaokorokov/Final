Команды для навигации и базовой работы с файлами

  • ls — покажет содержимое директории, в которой вы находитесь. Если после команды ввести адрес конкретной папки, то она покажет то, что хранится в ней;
  • cd — change directory, что в переводе означает «сменить директорию». Вписываем ее, а потом адрес: cd/users/test. Есть сокращенные варианты: cd- для перемещения в предыдущую директорию или cd ~ для перехода в домашнюю папку;
  • pwd — укажет полный путь до директории, в которой вы находитесь;
  • file — применяется для определения типа файла;
  • mv — применяется для перемещения файлов;
  • rm — команда для удаления файлов, директорий;
  • mkdir — создает новую директорию;
  • rmdir — удаляет пустую папку. Акцентируем внимание на слове «пустую». В отличие от rm, rmdir не трогает другие файлы и субдиректории;
  • cat — cоединить файлы в один и вывести их в стандартный вывод, имеется в виду связывание строк для вывода на экран.
