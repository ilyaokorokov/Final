Краткий справочник по командам командной оболочки (Cmd.exe)

  • chdir — показывает имя текущего каталога или выполняет смену текущего каталога. Если используется только буква диска (например, chdir C:), chdir отображает имена текущего каталога на указанном диске. При использовании без параметров chdir отображает текущий диск и каталог;
  • md — cоздает каталог или подкаталог. Расширения команд, которые включены по умолчанию, позволяют использовать одну команду md для создания промежуточных каталогов в указанном пути;
  • del (erase) — eдаляет один или несколько файлов.;
  • time — отображает или задает системное время. Если используется без параметров, время отображает текущее системное время и запрашивает ввод нового времени;
  • copy — копирует один или несколько файлов из одного расположения в другое;
  • dir — отображает список файлов и подкаталогов каталога;
  • echo — отображает сообщения или включает или отключает функцию эхо-команды. Команда echo удобнее при отключении эха . Чтобы отобразить сообщение с несколькими строками без отображения каких-либо команд, можно включить несколько echo команд после команды отключения в пакетную программу;
  • exit — выход из оболочки командной строки;
  • mkdir — создает подкаталог в текущем или заданном каталоге;
  • move — перемещает файл или группу файлов из текущего или заданного исходного каталога в указанный каталог;
  • rd (rmdir) — удаляет каталог или каталог вместе с его подкаталогами;
  • ren (rename) — переименовывает файл или группу файлов;
  • date — показывает или устанавливает системную дату;
  • type — показывает содержимое файла.
